# Code Rosetta

Cross-language codebase graph for LLM agents. Python, Terraform/HCL, YAML, Jinja2 — one unified map.

## Install

```bash
uv tool install code-rosetta-mcp
```

## Usage

```bash
code-rosetta build       # Parse all files into the graph
code-rosetta status      # Show graph statistics
code-rosetta serve       # Start MCP server
```
